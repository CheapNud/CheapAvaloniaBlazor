# CheapAvaloniaBlazor Templates

Project templates for [CheapAvaloniaBlazor](https://github.com/CheapNud/CheapAvaloniaBlazor) — cross-platform desktop apps with Blazor, Avalonia and Photino.

## Install

```
dotnet new install CheapAvaloniaBlazor.Templates
```

## Templates

| Short name | Description |
|------------|-------------|
| `cheapblazor` | Desktop app with MudBlazor UI |
| `cheapblazor-full` | Full-featured app: system tray, global hotkeys, notifications, settings persistence |

## Usage

```
dotnet new cheapblazor -n MyApp
cd MyApp
dotnet run
```

See the [documentation](https://github.com/CheapNud/CheapAvaloniaBlazor#readme) for details.
